<h1>zar_at_bot</h1>
<p>A Turkish Discord bot that rolls dices in a humoric way.</p>

<h2>Features</h2>
<ul>
    <li>Turkish language</li>
    <li>Easy setup and use</li>
    <li>Intended for a single server</li>
</ul>

<h2>Setup</h2>
<ol>
    <li>Clone the repository:
        <pre><code>git clone https://github.com/eard3n1/zar_at_bot.git</code></pre>
    </li>
    <br>
    <li>Install dependencies:
        <pre><code>pip install discord.py python-dotenv</code></pre>
    </li>
    <br>
    <li>Configure your bot token:
        <ul>
            <li>Rename the <code>.env.example</code> file to <code>.env</code></li>
            <li>Pass in your token inside <code>DISCORD</code></li>
        </ul>
    </li>
    <br>
    <li>Run the bot:
        <pre><code>python main.py</code></pre>   
    </li>  
</ol>
